# QuizMaster

a dynamic quiz application using PHP, HTML, CSS, and JavaScript with the
following features:
● Allow users to select from various quiz categories and different difficulty levels
(easy, medium, and hard) to customize their quiz experience.
● Implement a countdown timer for each question to limit the time available to
answer
● After completing the quiz, display a detailed result breakdown, including the
number of correct and incorrect answers, time spent per question, and overall
performance.
